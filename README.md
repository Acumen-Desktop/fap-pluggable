# Conveyor Evaluation App

This is an Electron app to evaluate **Conveyor** by [Hydraulic](https://www.hydraulic.dev):  
Build self-updating desktop app packages in minutes. Deploy to every OS from any OS. Platform native formats, fully signed and notarized.

## Overview
This project leverages the [pluggable-electron](https://github.com/dutchigor/pluggable-electron) framework, which allows for building Electron apps that can be extended by other parties.

## Features
- Self-updating desktop application packages.
- Cross-platform deployment capabilities.
- Native format support with signing and notarization.

## Installation
To install the app, follow these steps:
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd fap-pluggable
   ```
2. Install dependencies using your preferred package manager (npm, yarn, or pnpm).
   ```bash
   npm install  # or yarn install / pnpm install
   ```

## Usage
To run the app, execute:
```bash
npm start  # or yarn start / pnpm start
```
